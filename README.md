# whalerank

[GitHub](https://github.com/zjuchenyuan/whalerank)

Ranking the whales in staking pools.

Currently, only support USD pools.

Example: Currently, the biggest whale in Alpaca BUSD pool deposit 30.9 Million Dollars.
